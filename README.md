# Deep Learning Models for Anomaly Detection in Authentication Logs

Academic project made for the Deep Learning course at [Polytechnic University of Bari](https://www.poliba.it/).
The exam consisted at developing and comparing three different Deep Learning models (LSTM, Bi-LSTM with Attention, Transformer) aiming at performing anomaly detection in authentication logs. The dataset used for this project comes from [Los Alamos National Lab](https://csr.lanl.gov/data/auth/).
